## GlucoSense

Frontend-only React app to estimate postprandial glucose impact using Gemini API.

### Tech
- React + Vite
- Tailwind CSS
- Recharts
- No backend; localStorage only

### Setup
1) Install deps:
```
npm install
```
2) Create `.env` (optional; a fallback key is in code for quick start):
```
VITE_GEMINI_API_KEY=YOUR_KEY
```
3) Run dev:
```
npm run dev
```

### Notes
- This is not medical advice.
- API key on frontend is not secure; use for demo purposes only.
